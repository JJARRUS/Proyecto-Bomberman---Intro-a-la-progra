Entornos utilizados 

- Visual Code 
- IDLE 
- Git Bash 

Controles del Juego 
- W, A, S y D: Movimiento del personaje 
- X: Colocar Bomba
- 1, 2 Y 3: Items 
- R: Powerup de Corazón 
- T: Powerup de Daño 

Mecanicas Claves
- Destruir bloques para mostrar items, powerups y llave
- Usar bombas o habilidades para eliminar enemigos 

Notas Extra
- El juego tiene cuatro niveles con dificultad que aumenta por nivel
- Los corazones, bombas y velocidad se reinician por nivel
